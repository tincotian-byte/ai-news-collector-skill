# AI资讯收集器 (AI News Collector)

自动从网络搜索获取最新AI动态信息，整合国内外AI大厂及研究机构资讯。

## 功能特点

- **自动采集**: 从OpenAI、Anthropic、DeepSeek、阿里千问等国内外AI大厂获取最新动态
- **智能处理**: 自动去重、中文翻译、趋势总结
- **本地存储**: 按日期生成Markdown格式简报
- **可视化界面**: 蓝白配色的HTML阅读器，支持多维度筛选

## 使用方法

### 1. 收集今日AI资讯

直接对AI助手说：
- "收集今天的AI资讯"
- "获取最新AI动态"
- "生成AI新闻简报"

### 2. 查看资讯

收集完成后，打开生成的 `index.html` 文件即可浏览所有资讯。

### 3. 筛选功能

HTML阅读器支持以下筛选：
- **日期范围**: 选择起止日期
- **厂商**: 多选筛选特定厂商
- **技术领域**: 按领域标签筛选
- **优先级**: 高/中/低优先级

## 技术说明

### 去重策略
1. 基于标题相似度匹配
2. 基于URL唯一性校验
3. 24小时内相似内容自动合并

### 翻译策略
- 所有英文标题自动翻译为中文
- 提供中文摘要
- 保留原文链接
